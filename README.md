# Sedisson Co Website

A responsive, multi-page website for Sedisson Co, built around its commercial growth systems practice.

## Pages

- Homepage
- Commercial Growth Systems
- The Sedisson Method
- Solutions
- Selected Work
- Insights and long-form articles
- Contact and Commercial Readiness Assessment

## Run locally

Serve the directory with any static web server, then open `index.html` through that server.

```sh
python3 -m http.server 4173
```

The site uses plain HTML, CSS and JavaScript with no build step or external dependencies.
